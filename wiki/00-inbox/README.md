# 📥 Inbox

떠오르는 아이디어, 짧은 메모, "나중에 정리할 것"을 빠르게 던지는 곳.

## 규칙

- 파일명: `YYYY-MM-DD-짧은-슬러그.md`
- frontmatter는 `tags: [idea]` 정도만 있어도 됨
- 정리는 나중에 — 즉시 분류하려 하지 말 것
- 주기적으로 `/process-inbox` 커맨드 실행하면 LLM이 분류·이동·링크

## 예시 파일명

- `2026-04-26-onboarding-tooltip-아이디어.md`
- `2026-04-27-prompt-caching-개선-아이디어.md`
