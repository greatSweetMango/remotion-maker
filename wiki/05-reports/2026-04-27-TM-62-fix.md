# TM-62 — ParticlePhysics 중력+bounce 효과 강화

## TL;DR

TM-43 시각 검수에서 발견된 "particle-physics 입자가 바닥 일직선 정렬 + dramatic motion 부족" 이슈를 물리 파라미터 + 적분 루프 수정으로 해소. 입자가 중간/말미까지 다양한 높이로 튀고 좌우로 분산됨을 캡처로 검증.

## 무엇이 바뀌었나

`src/remotion/templates/ParticlePhysics.tsx`:

| 항목 | before | after | 의도 |
|---|---|---|---|
| `gravity` default | 1.4 | **1.8** | 더 강한 낙하 가속 |
| `gravity` scale (frame²) | × 0.6 | **× 0.85** | 60fps 기준 visible accel ↑ |
| `bounce` default | 0.78 | **0.86** | 반발 계수 ↑ → 재바운스 횟수 ↑ |
| `emitSpeed` default | 18 | **26** | 초기 분출 속도 ↑ |
| 분출 cone | ±54° (`0.6π`) | **±108° (`1.2π`)** | 좌우 fan-out 확장 |
| 초기 속도 분포 | `0.6 + r*0.8` | **`0.7 + r*0.9`** | 평균/분산 모두 ↑ |
| 바닥 마찰 vx | `× 0.92` | **`× 0.985`** | 슬라이드 유지 → "일직선 정렬" 방지 |
| 바닥 휴식 처리 | 없음 | **`abs(vy) < 0.6 ⇒ vy=0`** | 무한 micro-bounce 제거 (정착 자연) |

## 왜 / 배경

- TM-43 (`wiki/05-reports/2026-04-27-TM-43-template-visual-audit.md`)에서 spec "gravity-driven burst with bounce"인데 실제 렌더는 입자가 바닥 한 줄에 정렬되어 dramatic이 약함을 지적.
- 원인 분석:
  1. 중력 스케일 `× 0.6`이 60fps 환경에서 너무 약해 초기 상승 후 천천히 떨어짐.
  2. 바닥 마찰 `vx × 0.92`가 매 프레임 적용되어 수 바운스 안에 horizontal velocity가 0으로 수렴 → 모든 입자가 정확히 같은 높이(floor)에 쌓임.
  3. 분출 cone이 좁고 (`0.6π`) 거의 수직 방향만 → 처음부터 좌우 분산이 부족.
- 수정 후: gravity ↑ + bounce ↑ + 좁은 마찰 + 넓은 cone → 시간 경과에 따라 변화하는 비대칭 군집 패턴.
