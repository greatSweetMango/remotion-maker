## 라이브 wiring 검증 (TM-91 의 실제 사용자 표면)

라이브 hook (`useStudio`) + jsdom 키보드 이벤트로 다음을 모두 통과:

- `canUndo` / `canRedo` boolean 토글이 첫 편집 / 첫 undo 에 정확히 false↔true
- **Cmd+Z** → UNDO (preventDefault 적용)
- **Cmd+Shift+Z** → REDO
- **Cmd+Y** → REDO (Windows/Linux 익숙한 단축키 — 코드 라인 351 `key === 'y'`)
- **Ctrl+Z** → UNDO (cross-platform)
- 입력 필드(`<input>`) 포커스 시 핸들러 skip — 브라우저 내장 텍스트 undo 보존
- `CustomizePanel` Undo/Redo 버튼 클릭 → `onUndo`/`onRedo` 호출
- `canUndo=false` 시 Undo 버튼 `disabled`, `canRedo=false` 시 Redo 버튼 `disabled`
- `title` 속성으로 `Undo (Cmd+Z)` / `Redo (Cmd+Shift+Z)` 힌트 제공
