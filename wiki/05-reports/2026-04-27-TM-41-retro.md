- 카테고리별 stratified PRO sample (10건) 로 비용을 ~83% 절감하면서 카테고리 coverage 유지.
- 시각 캡처에서 의도하지 않았던 CSP `media-src` 글로벌 회귀를 잡아냄 (홈에서 175+ 에러).

## 잘 안 된 것 / 개선

- **acceptance 5초 p50 은 비현실적** 목표였을 가능성. gpt-4o-mini 비스트리밍이면 1k+ 토큰 응답에 8-12s 걸림. TM-54 가 streaming + parseProgressive 로 first-frame 재정의 필요.
- 시각 캡처를 진짜 60프레임 (1초 동영상) 으로 못함 — Player 가 정지 상태로 진입하고 customize UI 가 frame 0 만 보여줌. 추후 Playwright 의 `evaluate` 로 player.seekTo(N) 호출 + 스크린샷 시퀀스로 처리.
- task spec 의 "POST /api/generate" 풀 HTTP 경로를 거치지 못함 — 인증 우회 때문. usage limit / DB write 회귀는 본 회차에서 못 잡음.
- PRO tier 응답 raw 텍스트를 결과 JSON 에 보존하지 못함 (디버깅 시 빠른 분석 가능했을 것). 다음 회차에 raw response 필드 추가.
