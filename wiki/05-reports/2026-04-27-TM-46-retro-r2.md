## Lessons learned

1. **Phase 0 = `requires_env` 검증**. 인프라 작성 후 key 부재가 드러나면
   30분~1시간 sunk. 다음부터 spec 의 `requires_env` 를 받자마자
   `node -e "console.log(!!process.env.X && !!process.env.Y)"` 류 한 줄 점검.
2. **sandbox env strip**. `set -a; source .env.local; set +a` 가 child
   process 에 전달되지 않는 환경 발견. 모든 benchmark 스크립트는
   `dotenv.config()` 를 자체 부팅 단계에서 호출하도록 강제.
3. **renderer.openBrowser 의 stripped Page**. setCookie/screenshot 메서드가
   없지만 `_client().send(...)` 로 raw CDP 사용 가능. 의존성 추가 없이
   브라우저 자동화 → r3 이후에도 패턴 재사용.
4. **PlayerPanel useEffect TDZ**. `Component` 보다 위에서 `useEffect` 가
   참조하면 dev-mode 에서 ReferenceError. 항상 `useMemo` 결과 변수 *뒤* 에
   참조 effect 배치. 본 회차 한 번 hot-fix 함.
