# TM-46 Visual LLM-as-judge — 4축 채점 루브릭

> 이 루브릭은 `tm-46-judge.ts` 가 Claude Opus 4.7 (`claude-opus-4-7-20260101`) 에게
> 보내는 system prompt 의 정의다. 각 축은 1-10 정수.

## 축 1: 레이아웃 균형 (`layout`)

화면 구성 요소(텍스트/도형/차트)의 배치, 여백, 시각 무게 분배.

| 점수 | 기준 |
|---|---|
| 9-10 | 황금비/3분할 등 디자인 원칙 적용. 여백/정렬 의도적. 모든 요소가 안정적 위치. |
| 7-8  | 균형 양호하나 일부 요소가 가장자리에 과밀하거나 중앙 강박. |
| 5-6  | 한쪽으로 치우침 또는 빈 영역 과다. 그래도 가독 가능. |
| 3-4  | 텍스트 잘림, 요소 겹침, 전체적으로 불안정. |
| 1-2  | 화면 밖 요소, 의미 없는 배치, 빈 캔버스. |

## 축 2: 타이포 가독성 (`typography`)

폰트 크기, 색상 대비(WCAG AA 기준 4.5:1 이상 권장), 줄간격, 폰트 패밀리 일관성.

| 점수 | 기준 |
|---|---|
| 9-10 | 대비 7:1 이상, 위계 명확, 폰트 패밀리 1-2개 일관. 글자 짜임 정돈. |
| 7-8  | 대비 4.5:1 이상, 위계 인식 가능, 약간의 자간/행간 어색함. |
| 5-6  | 대비 3:1 근방, 일부 텍스트 작음. 읽으려면 노력 필요. |
| 3-4  | 대비 미달 (밝은 배경 위 흰 텍스트 등), 글자 잘림. |
| 1-2  | 텍스트 식별 불가. |

## 축 3: 모션 자연스러움 (`motion`)

3프레임(1초/중간/끝)을 연속으로 보고 보간/이징/타이밍을 추론.

- 1초 프레임은 도입(intro), 중간은 강조(hold/peak), 끝은 종료(outro)이 기대됨.
- spring/ease-out 계열 부드러움 우선. 선형 보간은 감점.
- 끝 프레임이 1초 프레임과 동일하면 (애니메이션 없음) 큰 감점.

| 점수 | 기준 |
|---|---|
| 9-10 | 3 프레임 차이가 자연스러운 진행. 의도적 강조점. 잔상/오버슛 살짝. |
| 7-8  | 진행은 보이나 단조로움 (선형). |
| 5-6  | 변화 미미. 1프레임만 살아있음. |
| 3-4  | 잘못된 방향(끝→시작 회귀) 또는 끊김. |
| 1-2  | 정지 프레임. 애니메이션 부재. |

## 축 4: 프롬프트 부합도 (`fidelity`)

원 프롬프트의 핵심 키워드(주제/색상/숫자/레이아웃)가 결과물에 반영되었는지.

| 점수 | 기준 |
|---|---|
| 9-10 | 주제+색상+숫자 모두 일치. 의도된 분위기까지. |
| 7-8  | 주제+색상 일치, 일부 디테일 누락. |
| 5-6  | 주제만 일치, 색상/숫자 다름. |
| 3-4  | 주제 부분 일치 (예: 차트 요청 → 텍스트만 나옴). |
| 1-2  | 무관한 결과물. |

## 종합 점수 계산

- 프레임당 4축 점수(1-10) → 0-100 으로 환산: `(layout + typography + motion + fidelity) * 2.5`
- 프롬프트당 점수: 3 프레임 평균
- 70 미만 → follow-up task spawn 대상. 75 이상 평균 → APPROVE.

## JSON 출력 강제

Judge 응답은 다음 스키마로 강제:

```json
{
  "frames": [
    {"frame": 60, "layout": 8, "typography": 7, "motion": 6, "fidelity": 9, "comment": "..."},
    {"frame": 90, "layout": ..., ...},
    {"frame": 180, "layout": ..., ...}
  ],
  "overall_comment": "한국어 1-2 문장",
  "improvement_suggestion": "프롬프트/템플릿/렌더 개선안 1-2 문장"
}
```

`max_tokens: 600` (3 frames × 4 axes + comments). `system` 에 본 루브릭 요약 + 출력 스키마 강제.
